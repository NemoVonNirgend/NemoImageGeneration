# Nemo Image Generation

Standalone automatic image workflow extracted from NemoPresetExt. It detects Pollinations image URLs and routes regeneration through SillyTavern's configured image-generation backend.

Install through Nemo Hub or with:

`https://github.com/NemoVonNirgend/NemoImageGeneration`

Reload after installation and configure an image provider in SillyTavern.
