# Nemo Image Generation

Standalone automatic image workflow for Pollinations URLs and SillyTavern's configured image-generation providers.

Install through Nemo Hub or SillyTavern's third-party extension installer with:

`https://github.com/NemoVonNirgend/NemoImageGeneration`

Its native Extensions settings drawer controls automatic handling, image style, and positive/negative prompt guidance. Settings persist in `extension_settings.NemoImageGeneration`; compatible former NemoPresetExt values migrate on first launch. Provider credentials remain managed by SillyTavern's image-generation extension.
